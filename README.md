## 📌 Overview

**TruthLens AI** is an intelligent **AI-powered fake news detection platform** that combines **Machine Learning**, **Deep Learning**, and **Graph Neural Networks (GNNs)** with a modern **React + Vite** web application to identify misinformation in real time.

The project is designed as a comprehensive AI research and deployment platform, enabling users to compare multiple models, analyze prediction results, and interact with an intuitive web interface.

Whether you're a **researcher**, **student**, **developer**, or **AI enthusiast**, TruthLens AI demonstrates an end-to-end workflow—from data preprocessing and model training to deployment through a responsive web application.

## 🤖 Machine Learning Models

Traditional machine learning algorithms are trained using text-based feature representations to establish strong baseline performance.

| Model | Purpose |
|--------|---------|
| Logistic Regression | Linear baseline for binary text classification |
| Naïve Bayes | Probabilistic text classification |
| Decision Tree | Rule-based classification |
| Random Forest | Ensemble learning with multiple decision trees |
| Support Vector Machine (SVM) | High-dimensional text classification |
| K-Nearest Neighbors (KNN) | Instance-based learning approach |

## 🧠 Deep Learning Models

Deep learning models automatically learn semantic and contextual representations from news articles.

| Model | Purpose |
|--------|---------|
| CNN | Extracts local textual patterns using convolutional filters |
| RNN | Learns sequential dependencies in textual data |
| LSTM | Captures long-term contextual information in news articles |

---

## 🌐 Graph Neural Network Models

Graph-based models leverage structural and relational information to improve fake news detection.

| Model | Purpose |
|--------|---------|
| GCN (Graph Convolutional Network) | Learns graph-based node representations |
| GraphSAGE | Aggregates neighborhood information for scalable learning |
| GAT (Graph Attention Network) | Uses attention mechanisms to focus on important graph relationships |

## ⚙️ Installation & Usage

Follow the steps below to set up **TruthLens AI** on your local machine.

---

## 📋 Prerequisites

Before running the project, make sure you have the following installed:

| Software | Version |
|----------|----------|
| Python | 3.10+ |
| Node.js | 18+ |
| npm | Latest |
| Git | Latest |

---

# 📥 Clone the Repository

```bash
git clone https://github.com/YOUR-USERNAME/TruthLens-AI.git
cd TruthLens-AI
```

---

# 🤖 AI Research Setup

Navigate to the AI Research directory.

```bash
cd AI-Research
```

Install the required Python packages.

```bash
pip install -r Requirements/requirements.txt
```

---

## 📂 Dataset

Place the dataset inside:

```text
AI-Research/
└── Datasets/
```

If required, update the dataset path in the notebooks before training.

---

## 🧠 Run the Notebooks

Launch Jupyter Notebook.

```bash
jupyter notebook
```

Open any notebook from:

```text
Machine-Learning/
Deep-Learning/
Graph-Neural-Network/
```

Run all cells to:

- preprocess data
- train the model
- evaluate performance
- save trained models

---

## 💾 Pretrained Models

Place the trained models inside:

```text
AI-Research/
└── Saved-Models/
```

If pretrained models are not available, simply execute the notebooks to generate them.

---

# 🌐 Web Application Setup

Navigate to the React application.

```bash
cd ../Web-App/client
```

Install dependencies.

```bash
npm install
```

Start the development server.

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```

---

## 🚀 Production Build

Generate an optimized production build.

```bash
npm run build
```

Preview the production build locally.

```bash
npm run preview
```

---

## 🧪 Using the Application

1. Open the Detect News page.
2. Enter or paste a news article.
3. Select an AI model.
4. Click **Analyze**.
5. View the prediction result.
6. Explore Analytics and Model Comparison pages.
