## 📁 Project Files

| File | Description |
|---|---|
| `Machine Learning.ipynb` | Machine Learning models — Logistic Regression, Random Forest, SVM |
| `Deep Learning.ipynb` | Deep Learning models — CNN, RNN, LSTM |
| `Graph Neural Networks.ipynb` | Graph Neural Network model implementation |
| `Exploratory_Data_Analysis.ipynb` | Data loading, cleaning, and visualization |

---

## ⚙️ How It Works

<div align="center">
 
| Step | Description |
|---|---|
| **📥 01 — Load & Explore** | Load and explore fake news datasets |
| **🧹 02 — Preprocess Text** | Cleaning, tokenization, and embedding |
| **📊 03 — Visualize Insights** | Exploratory Data Analysis and visualization |
| **🤖 04 — Train ML & DL Models** | Baseline models — SVM, LR, CNN, RNN, LSTM |
| **🕸️ 05 — Construct Graph** | Build graph representation for GNN input |
| **🧠 06 — Train & Evaluate GNN** | Train and evaluate on test data |
| **📈 07 — Compare Results** | Compare accuracy, F1-score across all models |
 
</div>

## 🚀 Getting Started

**1. Clone the repository**
```bash
git clone https://github.com/Kankana1012/Fake-News-Detection-Using-GNNs.git
cd fake-news-gnn
```

**2. Install required packages**
```bash
pip install pandas numpy matplotlib scikit-learn torch torch-geometric
```
